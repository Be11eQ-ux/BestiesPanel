# 闺蜜团使用手册 💕

> 这是你的三个「思维闺蜜」的使用指南。  
> 她们不会帮你做决定，但会把你的想法逼得更深、更清晰、更有创意。

---

## 她们是谁？

| 角色 | 定位 | 她的武器 |
|------|------|---------|
| 🌸 **Nana** | 天真追问者 | 只问问题，专门戳破你没注意到的假设 |
| ⚡ **Rye** | 激进怀疑者 | 挑战一切，帮你的论点变得无懈可击 |
| 🌿 **Chris** | 跨域联结者 | 从历史、科学、艺术找类比，给你全新视角 |

**重要：她们不会给你答案，只会让你越想越深。**

---

## 快速开始

### 第一步：设定时空背景

打开对话后，系统会先问你背景。选一个，或者自己写：

| 预设 | 适合讨论 |
|------|---------|
| 🏙️ 现代都市 | 职场、人际、日常决策 |
| 🏛️ 古代宫廷 | 权力、策略、人性 |
| 🚀 星际未来 | 技术伦理、长远影响 |
| 🧪 赛博朋克 | 数字、身份、反乌托邦 |
| 🧙 奇幻魔法 | 创意写作、世界观构建 |
| 🌿 田园乡村 | 生活方式、价值观 |
| ✏️ 自定义 | 随便你！比如「1990年代硅谷创业圈」 |

### 第二步：说出你的想法

背景设定好后，直接说出任何想法、观点、困惑或问题。

✅ **什么都可以说：**
- 一个你深信的观点（「我觉得努力就一定会成功」）
- 一个你做不了的决定（「我在考虑要不要辞职」）
- 一个刚冒出来的想法（「我觉得社交媒体让人更孤独」）
- 一个创意概念（「我想写一本关于……的书」）

## 对话节奏

### 每轮怎么运作？

每次你说完话，会有一位「本轮主角」（标有 ✨）充分展开，另外两人各补一句。这样你每次只需要消化一个主要角度，不会被三个方向同时轰炸。

主角每轮随机轮换，连续两轮不会是同一个人。

### 追问某一个人

每轮结尾都会出现提示：`💬 想继续追问谁？直接 @Nana / @Rye / @Chris`

直接用 `@名字` 点她：

```
@Nana 你的意思是……？
@Rye 但如果……的话呢？
@Chris 你说的那个历史案例，能展开说说吗？
```

只有被点名的人会完整回应，另两人不出现。

### 回到三人模式

```
继续
三个人都说说
换个话题：……
```

### 要求某种角度

```
@Rye 从最坏的情况说说
@Chris 有没有反直觉的历史案例？
@Nana 你觉得我的核心假设是什么？
```

---

## 读懂她们的回应

**Nana 的问题** 👉 不要急着回答，先想想为什么她会这样问——通常意味着你有个隐藏假设没想清楚。

**Rye 的挑战** 👉 她说的对立面不一定是真的，但值得认真对待。试着反驳她，看看你的论点是否站得住脚。

**Chris 的类比** 👉 她的跨领域连结有时候看起来很远，但试着顺着思路走——往往会到一个意外的地方。

## 常见用法场景

### 理清一个模糊的想法
说出你的直觉，让三人把它拆开检验。

### 做困难的决定
描述你的处境和选项，让 Rye 挑战你最倾向的那个选择，看看是真的想清楚了还是在自我安慰。

### 写作/创意构思
分享你的概念，让 Chris 用历史或科学类比帮你找到角度，让 Nana 问出你还没想清楚的部分。

### 辩论练习
说出你要辩护的立场，让 Rye 扮演对方，直到你能应对所有挑战为止。

## 她们有时候会自己聊起来

对话进行到一定程度，三位闺蜜可能会**忘记你的存在，开始自顾自地讨论**。

你会看到这样的分隔符：

```
〰️〰️〰️
（她们的小对话）
〰️〰️〰️
```

这是正常现象——话题足够热，或者某人刚才说了什么让另一人忍不住接嘴。这段插曲 2–3 轮后会自然结束，不用理她们，也不用等她们邀请——想说话直接插进来就好。

---

## 注意事项

- **她们不会给你答案**——这是设计，不是 bug。对话越深入，你对自己想法的掌握越清晰。
- **背景是装饰，不是限制**——你可以在「古代宫廷」背景下聊现代科技，反而会有意外的视角。
- **短回应不代表敷衍**——有时候一个精准的问题比长篇大论更有力。
- **可以随时换背景**——直接说「换成星际未来背景」即可，三人会自动重新校准。

```
暂时退出闺蜜模式
```

如果要重新进入：

```
回到闺蜜模式
```
